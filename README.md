# Animal Game
Animal game is an children friendly version of the traditional Rock Paper Scissor game wich is considered to be the oldest game in the world. The purpose of the game is to be simple and easy to understand for kids in every situation in life. The simplicity of the game will occupie parents kids when it's needed in different difficult situtions , like driving in the car or sitting in a buisness meeting. The game main focus points is colours, animation and simple buttons rather than popups with a lot of text wich make the game even usable for children with different types of handicaps.
